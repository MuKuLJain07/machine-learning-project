Student Score Predictor

Predict a student's average score using machine learning based on academic and demographic data.

Get Started

About the Project

This project is built using a machine learning model that takes input features such as:

  • Gender
  • Race/Ethnicity
  • Parental Level of Education
  • Lunch Type
  • Test Preparation Course
  • Total Score (out of 300)

Based on these inputs, the model predicts the student's average score. The backend is powered by Flask and the model is trained using Python libraries like Scikit-learn and Pandas. The UI is designed using HTML5, CSS3, and Bootstrap 5.
